Self-Check Quiz style


1. What is the HTML syntax to add an external JavaScript file?

Ans. To add an external JavaScript file to an HTML document, you can use the< script >element with the src attribute pointing to the URL or file path of the external JavaScript file. Replace "your-URL.js" with the actual URL or file path of your external JavaScript file. This script tag should typically be placed inside the < head > section of your html doument, although it can also be placed before the closing tag for better performance in some cases. This will include the external JavaScript file in your HTML document, allowing you to use the functions and code defined in that external script.

2. Can you run JavaScript in a file with a JS extension in your browser?

Ans. Yes, you can run JavaScript code stored in a file with a .js extension in your web browser.

3. How do you write a multiple-line comment in JavaScript?

Ans. In JavaScript, you can write multiple-line comments using the /* and */ delimiters. Anything enclosed between these delimiters will be treated as a comment and will not be executed by the JavaScript interpreter.

4. What is the best way to remove a line of code from running that you might want to keep as you debug?

Ans. To use a single comment by adding "//" on the starting of line of code is the best way to remove a line of code from running that you might want to keep as you debug.
